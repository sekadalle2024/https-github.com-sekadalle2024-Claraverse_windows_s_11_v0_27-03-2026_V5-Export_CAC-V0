🔄 FLUX DE TRAITEMENT
═══════════════════════════════════════════════════════════════════════════════

1. Message utilisateur → "Methodo revision" ou "Methodologie revision"
2. Router (Case 33) → http://localhost:5678/webhook/methodo_revision
3. Réponse n8n → JSON avec "Sous-section" / "Sub-items"
4. Détection FORMAT 7 → Génère __METHODO_REVISION_ACCORDION__[JSON]
5. MessageContentRenderer → Détecte le marqueur
6. Rendu → MethodoRevisionAccordionRenderer affiche l'accordéon
