================================================================================
TROUBLESHOOTING
================================================================================

Problème: Backend ne démarre pas
Solution:
  1. Vérifier conda: conda --version
  2. Vérifier l'environnement: conda env list
  3. Vérifier les logs: Receive-Job -Id 1 -Keep

Problème: Erreur d'import
Solution:
  1. Vérifier les fichiers: Test-Path "py_backend/tableau_flux_tresorerie_v2.py"
  2. Vérifier la syntaxe: python -m py_compile py_backend/tableau_flux_tresorerie_v2.py
  3. Redémarrer: .\stop-claraverse.ps1 puis .\start-claraverse-conda.ps1

Problème: Menu accordéon vide
Solution:
  1. Vérifier le fichier Excel a les bons onglets
  2. Vérifier les logs backend
  3. Exécuter le test: .\test-corrections-liasse.ps1

Problème: Colonnes N et N-1 manquantes
Solution:
  1. Vérifier que le fichier a Balance N et Balance N-1
  2. Vérifier que les patterns de détection sont corrects
  3. Vérifier les logs backend
