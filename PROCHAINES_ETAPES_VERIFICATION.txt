================================================================================
TROUBLESHOOTING
================================================================================

Problème: Menu accordéon vide
Solution:
  1. Vérifier que le fichier Excel a les bons onglets
  2. Vérifier les logs backend: Receive-Job -Id 1 -Keep
  3. Redémarrer le backend: .\stop-claraverse.ps1 puis .\start-claraverse-conda.ps1

Problème: Colonnes N et N-1 manquantes
Solution:
  1. Vérifier que le fichier a Balance N et Balance N-1
  2. Vérifier que les patterns de détection sont corrects
  3. Vérifier les logs backend

Problème: TFT vide
Solution:
  1. Vérifier que les balances N et N-1 sont présentes
  2. Vérifier les logs backend pour erreurs de calcul
  3. Exécuter le test: .\test-corrections-liasse.ps1

Problème: Annexes incomplètes
Solution:
  1. Vérifier que les données sont présentes dans les balances
  2. Vérifier les logs backend
  3. Exécuter le test: .\test-corrections-liasse.ps1
